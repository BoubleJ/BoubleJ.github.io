```
{props.on && <CheckIcon className={style.btnSrchIcon} />}

→ props.on 가 true면 <CheckIcon className={style.btnSrchIcon} /> 출력

props.on 가 false면 <CheckIcon className={style.btnSrchIcon} /> 미출력 → null
```
